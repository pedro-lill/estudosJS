Modelagem algebrica de uma funcao quadratica

Para a funcao quadratica f(x) = ax2 + bx + c, onde a, b e c sao numeros reais, temos:

Delta = b2 - 4ac

Se Delta > 0, a funcao tem duas raizes reais e distintas, sendo elas:

x1 = (-b + √Delta) / 2a

x2 = (-b - √Delta) / 2a

Se Delta = 0, a funcao tem duas raizes reais e iguais, sendo elas:

x1 = x2 = -b / 2a

Se Delta < 0, a funcao nao tem raizes reais, sendo elas:

x1 = (-b + √Delta) / 2a

x2 = (-b - √Delta) / 2a

Para a funcao quadratica f(x) = 2x2 - 3x + 1, temos:

Delta = (-3)2 - 4(2)(1)

Delta = 9 - 8

Delta = 1

Como Delta > 0, a funcao tem duas raizes reais e distintas, sendo elas:

x1 = (-(-3) + √1) / 2(2)

x1 = (3 + 1) / 4

x1 = 4 / 4

x1 = 1

x2 = (-(-3) - √1) / 2(2)

x2 = (3 - 1) / 4

x2 = 2 / 4

x2 = 1 / 2

Como Delta = 0, a funcao tem duas raizes reais e iguais, sendo elas:

x1 = x2 = -(-3) / 2(2)

x1 = x2 = 3 / 4

x1 = x2 = 3 / 4

x1 = x2 = 3 / 4

Como Delta < 0, a funcao nao tem raizes reais, sendo elas:

x1 = (-(-3) + √(-1)) / 2(2)

x1 = (3 + √(-1)) / 4
